# SAP-FI-End-to-End-Business-Process-Simulation-S-4HANA-fiori

## OVERVIEW
This project demonstrates an end-to-end financial accounting (FI) process executed in SAP S/4HANA using fiori applications.

The objective is to simulate real-world enterprise financial operations including master data creation,transection posting,and financial reporting.

## KEY PROCESS COVERED
- business partner creation 
- general ledger (G/L) accounting
- journal Entry posting
-Financial Statement reporting

## TOOLS & ENVIRONMENT
- SAP S/4HANA Cloud (Fiori Interface)
- SAP Fiori Apps 

## PROCESS FLOW
1. Create Business Partner (vendor/customer)
2. Maintain financial Data
3. post journal Entries 
4. Verify Transactions 
5. Display financial Statements
